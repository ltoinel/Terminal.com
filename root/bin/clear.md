  # CLEAR(1)

  ## NAME
  clear — clear the screen

  ## SYNOPSIS
  clear

  ## DESCRIPTION
  Clears the whole terminal and starts from a blank screen. The command
  history is not affected.

  ## EXAMPLES
  clear

  ## SEE ALSO
  history
